# Messagerie App

Application de messagerie style iMessage, construite avec React + Supabase + Netlify.

## Fonctionnalités
- Connexion / inscription par email
- Messages en temps réel
- Envoi de photos
- Groupes de discussion
- Statut lu/non lu

## Technologies
- React + Vite
- Supabase (base de données + temps réel + stockage)
- Netlify (hébergement)
